Пошаговое подключение к GrozNetVPN

1. Установите приложение WireGuard
- iPhone / iPad — через App Store
- Android — через Google Play
- Windows / macOS — с официального сайта WireGuard
2. Получите конфигурационный файл
Мы отправим вам .conf-файл после активации подписки. Этот файл нельзя передавать третьим лицам.
3. Импортируйте конфиг в WireGuard
В приложении WireGuard нажмите «Импорт», выберите файл.
4. Нажмите «Подключить»
Через 1–2 секунды VPN подключится.
